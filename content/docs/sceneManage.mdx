title: Quản lý view

## setDarkTheme

Hàm set Dark/Light theme cho viewer

```js
viewer.setDarkTheme(dark: boolean);
```

## saveViewCamera

Lưu lại góc nhìn của camera

```js
const saveViewer = viewer.saveViewCamera();
```

- `return` saveViewer với type là [IViewSave](sceneManage#lưu-view-camera)

## loadViewCamera

Set lại góc nhìn của camera

```js
const saveViewer = viewer.loadViewCamera(cameraView: IViewSave);
```

- cameraView với type là [IViewSave](sceneManage#lưu-view-camera)

## flyToHomeLocation

View tự động di chuyển về vị trí [setLocationHome](sceneManage#flytohomelocation) (Đối với trường hợp không gán giá trị cho setLocationHome thì động di chuyển về boundingBox của tất cả mô hình)

```js
viewer.flyToHomeLocation();
```

## setLocationHome

Lưu lại vị trí mặc định (Nếu không gán thì tự động tính toán về boundingBox của tất cả mô hình và lấy vị trí của nó)

```js
viewer.setLocationHome(longitude: number, latitude: number, height: number);
```

## flyToLocation

Di chuyển đến tọa độ

```js
viewer.flyToLocation(longitude: number, latitude: number, height?: number, duration?: number);
```
- default `height = 1000`
- default `duration = 2`

## destroyView

Hàm hủy View

```js
viewer.destroyView();
```

## IViewSave

| Prop       | Type                           | Note |
| ---------- | ------------------------------ | ---- |
| `heading`  | number                         | -    |
| `pitch`    | number                         | -    |
| `position` | `{x:number,y:number,z:number}` | -    |
| `roll`     | number                         | -    |
