## Các bước khởi tạo View

<div className='fd-steps [&_h3]:fd-step'>
 
### Lấy API Token
- Gọi API để lấy token, truyền vào clientId, clientSecret, scopes (DataRead)
- Phương thức **"POST"**
- URL: 
```js 
https://translateservicedev.corebim.com/authApi/User/GetToken
```
- Data: 
```js
{
  "clientId": " clientId ",
  "clientSecret": " clientSecret ",
  "scopes": [
    0
  ]
}
```
### Khởi tạo View
- Cài đặt thư viện:
```ts tab="npm"
npm i "@dp_unity/dpu-gis-viewer"
```
```ts tab="yarn"
yarn add "@dp_unity/dpu-gis-viewer"
```
- Khởi tạo thẻ chứa view:
```jsx
  <div style={{ height: '100%' }} id="DPU-gis-view" ref={cesiumViewerContainerRef}></div>
```
- Khởi tạo view:
```tsx
import { DPUService } from "@dp_unity/dpu-gis-viewer";

let service = new DPUService();
let viewerDPU = await service?.initView(option);

````
**`option`** của initView được lấy tại [Option initView](install#toolbaroptions-object)
</div>
## Ví dụ
```js

import { DPUService } from "@dp_unity/dpu-gis-viewer";

export default function Home() {
  const optionToolbar: ToolbarOptions = {
      labelPoint: true,
      flyCamera: true,
  };

  const service = new DPUService();
  const viewerDPU = await service?.initView("DPU-gis-view","token",optionToolbar,"vn");

  return (
    <div style={{ height: '100%' }} id="DPU-gis-view"></div>
  )
}
````

## Option initView

| Prop                 | Type                      | Default                                 | Note                                                 |
| -------------------- | ------------------------- | --------------------------------------- | ---------------------------------------------------- |
| `containerId`        | string                    | -                                       | Id của thẻ chứa View [idView](install#khởi-tạo-view) |
| `token`              | string                    | -                                       | Token được lấy ở [token](install#lấy-api-token)      |
| `inputOptionToolBar` | `ToolbarOptions \|\| {} ` | [ToolbarOptions](install#khởi-tạo-view) | Khởi tạo thanh công cụ                               |
| `language`           | "vn" \| "en"              | "vn"                                    | Ngôn ngữ mặc định                                    |

## ToolbarOptions object

Tùy chọn hiển thị thanh công cụ

| Prop                | Type    | Default | Note                                  |
| ------------------- | ------- | ------- | ------------------------------------- |
| `treeTileModel`     | boolean | `true`  | Hiển thị mô hình                      |
| `homeLocation`      | boolean | `true`  | Hiển thị nút di chuyển về mặc định    |
| `yourLocation`      | boolean | `true`  | Hiển thị nút vị trí                   |
| `mapWeather`        | boolean | `true`  | Hiển thị nút tùy chỉnh bản đồ         |
| `measure`           | boolean | `true`  | Hiển thị nút đo khoảng cách           |
| `labelPoint`        | boolean | `false` | Hiển thị nút chú thích điểm           |
| `dataProperties`    | boolean | `true`  | Hiển thị nút thông tin đối tượng      |
| `flyCamera`         | boolean | `false` | Hiển thị nút di chuyển theo lũy tuyến |
| `point`             | boolean | `true`  | Hiển thị nút tọa độ                   |
| `issuePoint`        | boolean | `false` | Hiển thị nút tọa độ chú thích         |
| `flyToVietNam`      | boolean | `true`  | Di chuyển mặc định về Việt Nam        |
| `requestRenderMode` | boolean | `true`  | Tự động render                        |
